【踊れるアニクラ】≠ODAYAKA 3rdMIXX #ノットオダヤカ 告知

19:20

Date

2020/12/12 (Sat)

Start 15:00(予定)

Place

アニソンDJ Bar 立川 Gluck Zwei

〒190-0023
東京都立川市柴崎町2丁目2-21
田園キャッスルビル立川 M1F
http://gluck-ltd.com/zwei

Entrance

¥2,000 (1Drink込み)


Reserve

新型コロナウイルス感染症対策のため本イベントでは下記フォームより参加表明登録をお願いしております。
ご協力お願いいたします。

【⬇︎参加表明申込フォームはこちら⬇︎】

≠ODAYAKA 3rdMIXX 参加申し込みフォーム - Googleフォーム


※入力頂いた情報は人数確認のため、また人数制限を超過した場合にご連絡させて頂くためのみに使用させて頂きます。

Stream

当日はTwitchにて配信を行います。
Gluck ZweiのTwitchチャンネルに是非ご登録ください。
https://www.twitch.tv/gluckzwei

Attention

・当日、会場の出入りは自由ですが会場近くでの大声での会話などは近隣の迷惑になりますので絶対におやめ下さい。
・20歳未満の方の飲酒・喫煙を固くお断りしております。
・会場は飲食店ですので、飲食物の持ち込みはご遠慮ください。
・当日身分証明書のご提示をして頂く場合がございますが、ご了承の程を宜しくお願い致します。
・貴重品の管理は、各自の責任において厳重に行ってください。店内における盗難、紛失等に関しましては一切責任を負いませんので、予めご了承下さい。
・他のお客様のご迷惑になる行為を発見した場合や、スタッフの指示に従っていただけない場合は、退場の措置を取らせていただくことがあります。

・新型コロナウイルス感染症予防のため下記の事項を実施致します。ご協力お願い致します。
1,検温
　当日必ず検温をしてお越しいただき、37.5度以上の場合は入場をご遠慮下さい。
　配信もございますのでそちらにてお楽しみ下さい。
2,消毒
　入場時に必ず消毒をお願い致します。
　また、室内設備の定期的な消毒作業にもご理解ご協力お願い致します。
3,三密への対策
　当日フロア内に人が多く、飛沫感染の可能性が高まる場合(いわゆる三密)入場制限を設けさせていただく可能性がございます。ご了承下さい。
4,マスク着用
　入場時には必ずマスクを着用下さい。

Cast

GUEST DJ

bassmicrobe  ( @bassmicrobe )

ネットレーベル「commune310」を主宰し、全国各地でのDJ出演や楽曲制作（アニソンREMIX、VTuberへの楽曲提供）、日々のプリコネ周回を中心に活動。

Letiamash   ( @Letiamash )

デレマスや音ゲー、SoundCloudなどに影響され、2017年に作曲活動を開始。
オリジナル曲やBootlegを作ったり作らなかったりする。
アニリミを中心としたプレーを好み、好きになったヒロインがよく負ける。

痺れた。( @_sibitto )
所属イベント: Linkage / リスト異常 / あにげん

DJをしたりアニソンのブートを作ったりしてます

t0k0r0( @tokorodokoro_05 )
所属イベント: Anistep(長崎) / カマクラ
東京在住。2010年頃からクラブ通いし、ただの客であったが、2017年に長崎転勤を期にDJ活動開始。
アニリミ中心でプレイを行う。
踊れるイベント大好きでユラユラするためにフロアやスピーカー前に張り付きがちのオタク。
Anistep(長崎)、カマクラ(東京)にレギュラーで参加中。


GUEST VJ


ダビドルイス  ( @occa_pd ) 
所属イベント: WAK!

仲間内のVJ不足により昨年よりVJの世界へ。
深刻な素材不足と難解なソフトウェアの板挟みになりながら日々奮闘中。
霧矢あおいさんと白鳥ひめさんが好き。


Regular VJ

Vjaith ( @anist0630 ) 
所属イベント:Anibeats
土佐脱藩型VJ ヤフ○クを駆使した映像に定評がある。


Regular DJ


おもちゃ ( @omotya17 )  

2019年に音楽喫茶茶箱の飲み会系イベント「san-sui」にてDJデビュー ゲーム楽曲やJ-POP、ハウス等オールジャンルな選曲を好む。 お世辞を真に受けることが得意であり、一番好きな言葉は「顔がいい」

DJ Asia Freaks ( @Asia_Freaks )
所属イベント: 救世主Ⅱ
立てば社畜
座ればオタク
歩く姿は田舎者

pokob  ( @pokob_ )

アニソンリミックスシーンとの出会いに影響を受け、2020年にDJキャリアをスタート。 以降ポップス・声優楽曲など、幅広いジャンルの音楽を展開しながら活動中。

チョリソ  ( @KaChoriso ) 
所属イベント: モアアニ / 顔面スプレッダーズ / 声優警察 / 音城セイクラ
2017年　師であるDJ異次元からコントローラーを受け取る。 2019年　上京した事をきっかけにDJデビュー。 2019年5月　DJクラウドと初主催イベント「MORE-ANI（モアアニ）」を開催。 幼少の頃から女児向けアニメの英才教育を受けて育ち、最近メンヘラとしての才能を開花させる。 ジャパニーズKawaiiフューチャーサウンドとジャパニーズウイスキーを好む。

Amana2 ( @Ama_na_2 )
2020年2月に #鮮やかなダンス 初回にてDJデビュー。 
以降クラブでお酒が飲める日を夢見ながらインターネットを中心にアニソンリミックス、UKベースミュージックなどを軸に活動中。

Kyomii ( @KNemaki ) 
所属イベント:Gluck Friday
キョミです。

うぇぼし〜 ( @_LiEwS )

2019年11月よりDJ活動を開始。
強い音楽とキラキラした音楽が好きな田舎産まれの貧弱DJ。
東京 立川で踊れるアニクラを目指し「≠ODAYAKA」を主催。
